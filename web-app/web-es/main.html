EverySecond

A digital short-term memory for the elderly

The Problem

01

With aging comes short-term memory loss. In normal cases, memory loss causes problems such as misplacing objects, forgetting important dates, and forgetting important habits.

02

On a more severe scale, seniors with memory-related conditions (such as Alzheimer’s) struggle with remembering close relatives, taking medicines at the correct time, eating healthy meals regularly, or even instances of elder abuse.

03

Many family members who live far away from their elderly relatives wish to check on the well-being of their relatives and ensure their physical and mental safety.

The Solution

EverySecond solves the problems by quickly retrieving "memories" for the elderly and their loved ones. To do this, EverySecond has three key features: it sees what the user sees, understands the images by using artificial intelligence, and recalls daily experiences by indexing these experiences into a datastore.

Device

First, EverySecond records what everything that the user sees. The device is wearable, so it captures data at any time and any place, just like the user’s eye. Each second, the device takes a picture, not missing a single moment.

Cloud

Secondly, EverySecond understands what the user sees. Incorporating AI from the Google Cloud Platform, EverySecond interprets the objects it records. It defines medicine, people, and food, which are the key components required to help an elderly with his or her accommodations. Then, a cloud function stores these memories inside a datastore so that EverySecond quickly responds to a request for a specific memory.

Web App

Finally, EverySecond recalls previous memories. The innovative image processing and indexing technique adds a layer of important data to the video and allows users to quickly find actions or objects witnessed in the video. Users access their digital memory through a friendly web app interface, which displays a carousel of relevant images for the user to easily verify the occurrences of an event and reduce false positives.
